# KeeperAgent
Implementation of keeperKit- an SDK built on top of Keeperhub. Built for OpenAgents EthGlobal. 
This agent is a self sufficient agent that can run and execute the commands given to it in plain english on KeeperHub via the API key provided to it. This agent takes two api keys - a gemini api key and a keeperhub api key which can be seen in the .env.example files. The agent once supplied with these keys is then self sufficient in executing all the features that our SDK provides. 
The agent is fully working, and the server is implemented in Typescript. The agent is very capable.

#KeeperUniversity 
This is a consumer app built on top of our SDK KeeperKit for KeeperHub. It is simply an AI workflow manager, where admistrative staff or students of a univeristy can built and execute personalized workflows. They can chat with our agent in plain english and describe the wrokflows they would want to build.
